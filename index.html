Спецальные символы и символьные операторы

.
Текущая директория
..
Директория на уровень выше
~
Домашняя директория
*
Любое количество символов
?
Ровно один любой символ
Программа < файл
Брать исходные данные (stdin) для выполнение программы из файла
Программа > файл
Выподить результат выполения программы (stdout) из файла (каждый запуск команды будет перезаписывать файл)
Программа 2> файл
Выподить ошибочный результат выполения программы (stdout) из файла (каждый запуск команды будет перезаписывать файл)
Программа >> файл
Выподить результат выполения программы (stdout) из файла (каждый запуск команды будет дополнять файл)
Программа 2>> файл
Выподить ошибочный результат выполения программы (stdout) из файла (каждый запуск команды будет дополнять файл)
Прогр1 | Прогр2 | ... | ПрогрN
Конвейер (pipe). Передает результат выполнения (stdout) Прогр1 в качестве входных параметров (stdin) для Прогр2 и так далее по всей цепочке

Команды

pwd
Вывод на экран адрес директории где сейчас находится пользователь
man команда
Вывод на экран справки о команде (выход Q)
clear
Очистка экрана
exit
Закрыть терминал
ls опции путь
Выведет на экран содержимое директории
-a (--all)
Отобразить все файлы
-h (--human-readable)
Файлы отобразятся в читаемом для человека виде
-l
Добавит информацию правах доступа и о размерах файлов
cd путь
Перейти в директорию
mkdir опции путь
Создать директорию
-p
Создать директорию и ее родителей
touch путь
Создать файл
rm опции путь
Удалить файл
-r
Удалить директорию
-rf
Удалить директорию без вопросов
cp опции путь_1 путь_2
Скопировать файл, расположенный по пути_1 в файл расположенный по пути_2
-r
Копировать директорию расположенную по пути_1 в директорию расположенную по пути_2
mv путь_1 путь_2
Переместить директорию или файл расположенную по пути_1 в директорию расположенную по пути_2
fg
Продолжить выполнение ранее запущенной программы (используется в терминале после сочетания клавиш Ctrl + Z для возврата программы в "активное" состояние)
bg
Перевести выполнение ранее запущенной программы в фоновый режим. Т.е. выполнение программы продолжиться, при этом в терминале можно будет запускать другие программы и терминал становится отвязан от программы (используется после сочетания клавиш Ctrl + Z)
cat путь_до_файла
Вывести содержимое файла на экран
less путь_до_файла
Открыть файл на чтение (q - выход, / - поиск, g - в начало, G - в конец)
nano путь_до_файла
Редактировать файл (Ctrl + X - выход)
wget опции путь ссылка
Скачать файл (в опциях задаются параметры/настройки для загрузки) по ссылке и расположить его в директории, указанной в пути
-P путь ссылка
Опция указывающая путь до диреткории, в которую необходимо положить файл загруженный по ссылке
-О путь ссылка
Опция указывающая путь до диреткории, а также название, под которым необходимо сохранить файл, загруженный по ссылке
-c ссылка
Докачать файл по ссылке в случае обрыва связи
--spider ссылка
Проверить доступность файла по ссылке
-i тестовый_файл
Скачать несколько файлов по ссылкам из текстового файла
-r-l глубина ссылка
Рекурсивное скачивание файлов по ссылке на указанную глубину (по умолчанию глубина 5)
-r-A тип,тип,...,тип ссылка
Рекурсивное скачивание файлов только опреденного типа (типов)
unzip архив.zip
Распаковать содержимое архива.zip
zip архив.zip файл1 файл2 ...
Запаковать перечисленные файлы и/или папки в архив.zip
gunzip архив.gz
Распаковать содержимое архива.gz, при этом после распаковки архива файл архив.gz удалиться
gzip файл
Запаковать файл в файл.gz, исходный файл удалить
tar опции архив.tar файл1 файл2 ...
Запаковать перечисленный файлы и/или папки в архив.tar без сжатия
-c
Опция создания архива
-v
Опция вывода информации на экран
-f
Опция сообщающая программе что создаваться будет файл на файловой системе (всегда в конце)
-z архив.tar.gz файл1 файл2 ...
Опция которая сразу архивирует перечисленные файлы и/или папки в архив.tar.gz (с сжатием при помощи gzip)
-x архив.tar
Опция которая распакует архив.tar
-xz архив.tar.gz
Опция которая сразу распакует архив.tar.gz с использованием gunzip
find <папка> -name "<имя файла>"
Поиск файлов в файловой системе по имени (по части имени)
grep опции "<строка>" <файл>
Поиск внутри файла строки с опциями или без
-c
Опция которая посчитает количество вхождений строки
-r/dt>
Опция которая даст команду программе производить поиск во всех файлах в папке
ssh логин@адрес_сервера опции
Вход по SSH на сервер
-p
Опция сообщающая порт подключения (не обязательно)
scp опции логин@адрес_сервера:путь1 путь2
Копирование файлов с сервера (путь1) на клиента (путь2)
-P
Опция сообщающая порт подключения
scp опции путь1 логин@адрес_сервера:путь2
Копирование файлов с клента (путь1) на сервер (путь2)
-P
Опция сообщающая порт подключения

Горячие клавиши

Tab
Показать возможные варианты завершения набраной части команды. Двойное нажатие приведет к выбору первого значения из предложенных вариантов
Ctrl + Alt + t
Запуск терминала в графической оболочке
Ctrl + C
Прервать выполнение текущей программы
Ctrl + Z
Приостановить выполнение текущей программы
Ctrl + Shift + C
Скопировать выделенный текст (аналог Ctrl + C)
Ctrl + Shift + V
Скопировать выделенный текст (аналог Ctrl + V)
Ctrl + A
Перейти в начало сроки команды
Ctrl + E
Перейти в начало сроки команды
